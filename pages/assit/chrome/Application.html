window 快速找到已经下载好扩展程序所在的位置

1、地址栏输入chrome:version 回车

2、用资源管理器打开"个人资料路径"栏的路径,该路径下的Extensions文件夹即默认的扩展安装路径

3、找到对应扩展程序的id，对应文件夹的名称即可

1、JSONView

  在网页里面格式化 JSON 数据，方便预览

2、Adobe Acrobat

  将当前网页转换为 Adobe PDF 文件

3、Vue.js devtools

  一款必备 Vue 的开发调试工具，可查看组件，vuex，路由等

4、React Developer Tools

  一款必备 React 的开发调试工具

5、Tampermonkey

  强大的油猴工具，结合 jS 脚本创造无限的可能

6、Lighthouse

  分析网页优化工具，帮助你优化网页性能

7、沉浸式翻译

  网页双语翻译，能在网页英语段落下面显示中文翻译

8、草料二维码

  方便手机扫码跳转网址，不用手动输入

9、iTab新标签页

  自定义浏览器的标签页，

10、SuperCopy 超级复制

  一键破解禁止右键、破解禁止选择、破解禁止复制、破解禁止粘贴，启用复制，启用右键，启用选择，启用粘贴。

11、图片助手(ImageAssistant) 批量图片下载器

  一款用于嗅探、分析网页图片并提供批量下载等功能的浏览器扩展程序。

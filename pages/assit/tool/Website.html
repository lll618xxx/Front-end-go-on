# 2.VSCode插件

  • Chinese Language - 设置VS中文语言
  • Vetur - Vue语法高亮、智能提示
  • View in Broswer - 右击html打开浏览器
  • Prettier - 代码格式化
  • Code Runner - 代码在VS运行
  • Git History - 查看文件修改历史
  • GitLens - 显示每一行的commit
  • Auto Close Tag - 自动闭合标签
  • Auto Rename Tag - 重命名标签时，闭合标签也会被重命名
  • Bracket Pair Colorizer - 用不同的颜色标识不同的括号，方便括号的识别
  • Error Gutters - 在异常代码行前面标记X号，方便查找错误
  • Todo Tree - 高亮的功能,初始的话,只支持TODO,和FIXME两个标记
  • Code Spell Checker - 英文拼写检测 （不建议用）
  • Document This - 可以快速地帮你生成函数注释 将光标放置于function上面，快捷键是 Ctrl+Alt+D
  • vscode-fileheader -  koroFileHeader 可以快速地文件头部注释 Ctrl+Alt+I
  • javascript console utils - 快速生成console.log Ctrl+Shift+L
  • Import Cost - 资源引入体积

# 2-1.设置中文语言

（1）使用快捷键组合【Ctrl+Shift+p】，在搜索框中输入“configure display language”，回车；
（2）修改locale.json文件下的属性“locale”为“zh-CN”;
（3）重启vscode；

​如果重启后仍然是英文显示，在商店中查看已安装的插件，
搜索Chinese（Simplied） Lang重新安装，然后再重启vscode

# 2-2.单击文件预览取消（原来文件被替换）

​利用快捷键Ctrl+Shift+P， 搜索settings(json)进入settings.json文件

"workbench.editor.enablePreview": false

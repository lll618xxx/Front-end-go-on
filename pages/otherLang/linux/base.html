# 1、基本命令

  • 查找某个文件位置的方法
find / -name nginx.conf
1
  • 查看端口使用情况
netstat -apn|grep 3000
1
  • get请求查看网址是否能访问
curl https://www.baidu.com
1
  • 使用ifconfig查看自己的IP地址是否含有IPv6地址
ifconfig
1
  • 将整个系统所有接口包括回环接口禁用IPv6，使用以下命令，但重启会失效，不是永久
sudo sh -c 'echo 1 > /proc/sys/net/ipv6/conf/all/disable_ipv6'
1

# 2、linux vim编辑

进入文件，vim nginx.conf
按 i 键，进入编辑模式，状态栏提示为“插入”
编辑完成后按 ESC 键退出编辑模式，状态栏的插入已没有
按键盘 :再输入wq，保存文件并退出
如果输入内容后，输入：q!，为直接退出不保存内容

# 1.什么是TypeScript

Typescript 是一个强类型的 JavaScript 超集，支持ES6语法，支持面向对象编程的概念，如类、接口、继承、泛型等。
Typescript并不直接在浏览器上运行，需要编译器编译成纯Javascript来运行。

# 2.TS和JS比较

1、TS是静态类语言，更加严谨规范；JS是动态类语言相对更灵活。 2、TS类型可以一定程度上充当文档

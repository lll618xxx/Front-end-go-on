1. жөҸи§ҲеҷЁзҺҜеўғVSе°ҸзЁӢеәҸзҺҜеўғ

еҫ®дҝЎе°ҸзЁӢеәҸжҳҜд»ӢдәҺNativeе’Ңweb appд№Ӣй—ҙзҡ„дә§зү©гҖӮе®ғдҫқжүҳжөҸи§ҲеҷЁпјҲwebviewпјүеұ•зӨәпјҢеҗҢж—¶еҸҜд»Ҙи°ғз”ЁеҺҹз”ҹиғҪеҠӣпјҲеҰӮиҺ·еҸ–йҖҡдҝЎеҪ•пјҢжӢҚз…§зӯүзӯүпјүпјҢеҗҢдёҖд»Ҫд»Јз ҒеҸҜиҝҗиЎҢеңЁAndroidпјҢiOSе’Ңеҫ®дҝЎи°ғиҜ•ејҖеҸ‘е·Ҙе…·еҶ…пјҲи·Ёе№іеҸ°иғҪеҠӣпјүгҖӮ

дёҺRNзҡ„и·Ёе№іеҸ°дёҚеҗҢпјҢе°ҸзЁӢеәҸеӨ§йғЁеҲҶUIз»„д»¶е№¶дёҚжҳҜеҺҹз”ҹжёІжҹ“пјҢиҝҳжҳҜзұ»дјјweb appз”ЁжөҸи§ҲеҷЁжёІжҹ“гҖӮеҸӘжңүе°‘йҮҸз»„д»¶жҳҜNativeе®һзҺ°пјҲNativeз»„д»¶еұӮеңЁWebViewеұӮд№ӢдёҠпјү

жөҸи§ҲеҷЁиҝҗиЎҢзҺҜеўғ

йҰ–е…ҲпјҢжөҸи§ҲеҷЁзҡ„дё»иҰҒз»„д»¶жңүпјҡ

  • з”ЁжҲ·з•Ңйқў(User Interface) - ең°еқҖж ҸгҖҒеүҚиҝӣ/еҗҺйҖҖжҢүй’®гҖҒд№ҰзӯҫиҸңеҚ•зӯүпјҲйҷӨдәҶжөҸи§ҲеҷЁдё»зӘ—еҸЈеӨ–пјҢе…¶д»–жҳҫзӨәзҡ„еҗ„дёӘйғЁеҲҶйғҪеұһдәҺз”ЁжҲ·з•ҢйқўпјүгҖӮ
  • жөҸи§ҲеҷЁеј•ж“Һ(Browser engine) - еңЁз”ЁжҲ·з•Ңйқўе’Ңе‘ҲзҺ°еј•ж“Һд№Ӣй—ҙдј йҖҒжҢҮд»ӨгҖӮ
  • е‘ҲзҺ°еј•ж“Һ(Rendering engine) important- иҙҹиҙЈжҳҫзӨәиҜ·жұӮзҡ„еҶ…е®№пјҲеҰӮжһңиҜ·жұӮзҡ„еҶ…е®№жҳҜ HTMLпјҢе®ғе°ұиҙҹиҙЈи§Јжһҗ HTML е’Ң CSS еҶ…е®№пјҢе№¶е°Ҷи§ЈжһҗеҗҺзҡ„еҶ…е®№жҳҫзӨәеңЁеұҸе№•дёҠпјүгҖӮ
  • JavaScript и§ЈйҮҠеҷЁ(JavaScript Interpreter) important - з”ЁдәҺи§Јжһҗе’Ңжү§иЎҢ JavaScript д»Јз ҒгҖӮ
  • зҪ‘з»ң(Networking) - з”ЁдәҺзҪ‘з»ңи°ғз”ЁпјҢжҜ”еҰӮ HTTP иҜ·жұӮгҖӮе…¶жҺҘеҸЈдёҺе№іеҸ°ж— е…іпјҢе№¶дёәжүҖжңүе№іеҸ°жҸҗдҫӣеә•еұӮе®һзҺ°гҖӮ
  • ж•°жҚ®еӯҳеӮЁ(Data Persistence)гҖӮиҝҷжҳҜжҢҒд№…еұӮгҖӮжөҸи§ҲеҷЁйңҖиҰҒеңЁзЎ¬зӣҳдёҠдҝқеӯҳеҗ„з§Қж•°жҚ®пјҢдҫӢеҰӮ CookieгҖӮж–°зҡ„ HTML и§„иҢғ (HTML5) е®ҡд№үдәҶвҖңзҪ‘з»ңж•°жҚ®еә“вҖқпјҢиҝҷжҳҜдёҖдёӘе®Ңж•ҙпјҲдҪҶжҳҜиҪ»дҫҝпјүзҡ„жөҸи§ҲеҷЁеҶ…ж•°жҚ®еә“гҖӮ
  • з”ЁжҲ·з•ҢйқўеҗҺз«Ҝ(UI Backend) - з”ЁдәҺз»ҳеҲ¶еҹәжң¬зҡ„зӘ—еҸЈе°ҸйғЁд»¶пјҢжҜ”еҰӮз»„еҗҲжЎҶе’ҢзӘ—еҸЈгҖӮе…¶е…¬ејҖдәҶдёҺе№іеҸ°ж— е…ізҡ„йҖҡз”ЁжҺҘеҸЈпјҢиҖҢеңЁеә•еұӮдҪҝз”Ёж“ҚдҪңзі»з»ҹзҡ„з”ЁжҲ·з•Ңйқўж–№жі•гҖӮ

дёҖиҲ¬жқҘиҜҙпјҢжөҸи§ҲеҷЁиҝҗиЎҢеңЁдёҖдёӘиҝӣзЁӢдёӯпјҲдҪҶжҳҜchromeжҜ”иҫғзү№ж®ҠпјҢжҜҸдёӘж ҮзӯҫйЎөйғҪжҳҜдёҖдёӘзӢ¬з«ӢиҝӣзЁӢпјүгҖӮеҗҢж—¶пјҢжөҸи§ҲеҷЁжҳҜеӨҡзәҝзЁӢзҡ„пјҢжҜ”иҫғйҮҚиҰҒзҡ„зәҝзЁӢжңүпјҡ

  • е‘ҲзҺ°еј•ж“ҺпјҲеҸҲз§°дёәжёІжҹ“еј•ж“ҺпјүпјҡиҝҗиЎҢеңЁUIзәҝзЁӢдёӯгҖӮ
  • JavaScript и§ЈйҮҠеҷЁпјҲеҸҲз§°дёәJSи§Јжһҗеј•ж“ҺпјүпјҡиҝҗиЎҢеңЁJSеј•ж“ҺзәҝзЁӢдёӯгҖӮ

жіЁж„ҸпјҡUI жёІжҹ“зәҝзЁӢдёҺ JavaScript еј•ж“ҺзәҝзЁӢдёәдә’ж–Ҙзҡ„е…ізі»пјҢеҪ“ JavaScript еј•ж“ҺзәҝзЁӢжү§иЎҢж—¶ UI жёІжҹ“зәҝзЁӢдјҡиў«жҢӮиө·пјҢUI жӣҙж–°дјҡиў«дҝқеӯҳеңЁдёҖдёӘйҳҹеҲ—дёӯзӯүеҲ° JavaScript еј•ж“ҺзәҝзЁӢз©әй—Іж—¶з«ӢеҚіиў«жү§иЎҢгҖӮ

е°ҸзЁӢеәҸиҝҗиЎҢзҺҜеўғ

е°ҸзЁӢеәҸиҝҗиЎҢж—¶дјҡжңүдёӨдёӘзәҝзЁӢпјҡгҖҺView ThreadгҖҸ е’Ң гҖҺAppService ThreadгҖҸпјҢзӣёдә’йҡ”зҰ»пјҢйҖҡиҝҮжЎҘжҺҘеҚҸи®®WeixinJsBridageиҝӣиЎҢйҖҡдҝЎпјҲеҢ…жӢ¬ setData и°ғз”ЁгҖҒcanvasжҢҮд»Өе’Ңеҗ„з§ҚDOMдәӢд»¶пјү

зәҝзЁӢеҗҚз§°жүҖеұһжЁЎеқ—иҝҗиЎҢд»Јз ҒеҺҹзҗҶеӨҮжіЁ
Viewи§ҶеӣҫеұӮпјҲеҸҜиғҪжңүеӨҡдёӘпјүWXML/WXSSwebviewжёІжҹ“wxmlзј–иҜ‘еҷЁжҠҠwxmlж–Үд»¶иҪ¬дёәjs(жһ„е»әvirtual dom);wxssзј–иҜ‘еҷЁжҠҠwxssж–Үд»¶иҪ¬еҢ–дёәjs
AppServiceйҖ»иҫ‘еұӮпјҲдёҖдёӘпјүJSJavascriptCoreиҝҗиЎҢж— жі•и®ҝй—® window/documentеҜ№иұЎ

жҸҗзӨә

дёӨдёӘзәҝзЁӢзӣҙжҺҘеҰӮдҪ•иҝӣиЎҢж•°жҚ®дј йҖ’пјҹеҸҜеҸӮиҖғеҫ®дҝЎе®ҳж–№иҜҙжҳҺпјҡ

йҖҡиҝҮдёӨиҫ№жҸҗдҫӣзҡ„ evaluateJavascript жүҖе®һзҺ°гҖӮеҚіз”ЁжҲ·дј иҫ“зҡ„ж•°жҚ®пјҢйңҖиҰҒе°Ҷе…¶иҪ¬жҚўдёәеӯ—з¬ҰдёІеҪўејҸдј йҖ’пјҢеҗҢж—¶жҠҠиҪ¬жҚўеҗҺзҡ„ж•°жҚ®еҶ…е®№жӢјжҺҘжҲҗдёҖд»Ҫ JS и„ҡжң¬пјҢеҶҚйҖҡиҝҮжү§иЎҢ JS и„ҡжң¬зҡ„еҪўејҸдј йҖ’еҲ°дёӨиҫ№зӢ¬з«ӢзҺҜеўғ

д№ҹе°ұжҳҜиҜҙпјҢдёӨдёӘгҖҺжЁЎеқ—/зәҝзЁӢгҖҸжҳҜйҖҡиҝҮзі»з»ҹеұӮзҡ„JSBridageжқҘйҖҡдҝЎзҡ„пјҢйҖ»иҫ‘еұӮжҠҠж•°жҚ®еҸҳеҢ–йҖҡзҹҘеҲ°и§ҶеӣҫеұӮпјҢи§ҰеҸ‘и§ҶеӣҫеұӮйЎөйқўжӣҙж–°пјҢи§ҶеӣҫеұӮжҠҠи§ҰеҸ‘зҡ„дәӢд»¶йҖҡзҹҘеҲ°йҖ»иҫ‘еұӮиҝӣиЎҢдёҡеҠЎеӨ„зҗҶгҖӮеҸҜеҸӮиҖғдёӢеӣҫпјҡ

жүҖд»ҘеҸҜд»Ҙеҫ—еҮәеҰӮдёӢз»“и®әпјҡ

  • е°ҸзЁӢеәҸjsд»Јз Ғж— жі•ж“ҚдҪңDOMеҜ№иұЎпјҢд№ҹж— жі•зӣҙжҺҘж“ҚдҪңwxmlдёҠзҡ„е®№еҷЁжҲ–з»„д»¶пјҲjsд»Јз Ғе’ҢwebviewжІЎжңүиҝҗиЎҢеңЁеҗҢдёҖдёӘзәҝзЁӢдёӯпјү

  • еҰӮжһңйңҖиҰҒеңЁView ThreadдёӯиҝҗиЎҢиҮӘе®ҡд№үjsд»Јз ҒпјҢеҸҜд»ҘдҪҝз”ЁwxsпјҲеҫ®дҝЎејҖеҸ‘зҡ„и„ҡжң¬иҜӯиЁҖпјүпјҢе®ғе’ҢViewеҗҢдёҖдёӘзәҝзЁӢгҖӮ

е№іеҸ°жёІжҹ“jsиҝҗиЎҢзҺҜеўғ
iOSWKWebViewжёІжҹ“пјҲзҺҜеўғжңү iOS8гҖҒiOS9гҖҒiOS10пјүJavaScriptCore
AndroidX5 еҹәдәҺ Mobile Chrome 37 еҶ…ж ёжқҘжёІжҹ“X5 JSCoreжқҘи§Јжһҗ
ејҖеҸ‘е·Ҙе…·Chrome Webview жёІжҹ“nwjs дёӯ

иҷҪз„¶зӣ®еүҚе°ҸзЁӢеәҸдҪҝз”Ё webview жёІжҹ“пјҢдҪҶжҳҜдёҚж„Ҹе‘ізқҖе®ғд»ҘеҗҺд№ҹдёҖзӣҙдҪҝз”ЁwebviewжёІжҹ“гҖӮдҪңдёәејҖеҸ‘иҖ…пјҢеҸӘиғҪдҫқиө–е°ҸзЁӢеәҸжҸҗдҫӣзҡ„зҺҜеўғгҖӮиҖҢиҝҷдёӘзҺҜеўғеҶҚдёӢеұӮеҰӮдҪ•еӨ„зҗҶпјҢе№¶дёҚеҸ—ејҖеҸ‘иҖ…жҺ§еҲ¶пјҢиҝҷж„Ҹе‘іе°ҸзЁӢеәҸжңӘжқҘеҫҲеҸҜиғҪе…ЁйқўйҮҮз”ЁеҺҹз”ҹжёІжҹ“пјҢзұ»дјјRNжҲ–WeexпјҢжҜ•з«ҹпјҢеҺҹз”ҹзҡ„UIдҪ“йӘҢжӣҙеҘҪгҖӮ

2. е°ҸзЁӢеәҸеә•еұӮеҺҹзҗҶдёҺжҠҖжңҜи§Јжһҗ(вӯҗпёҸ)

е°ҸзЁӢеәҸзҡ„иҝҗиЎҢжңәеҲ¶д№ӢжүҖд»ҘиғҪеӨҹеңЁе®ҝдё»еә”з”ЁпјҲеҰӮеҫ®дҝЎгҖҒж”Ҝд»ҳе®қзӯүпјүеҶ…йғЁй«ҳж•ҲиҝҗиЎҢпјҢе№¶дё”жӢҘжңүзӢ¬з«Ӣзҡ„иҝӣзЁӢпјҢдё»иҰҒдҫқиө–дәҺе…¶зӢ¬зү№зҡ„жһ¶жһ„и®ҫи®Ўе’Ңж“ҚдҪңзі»з»ҹзә§зҡ„иө„жәҗйҡ”зҰ»гҖӮ

дёҖ. дёәд»Җд№ҲиғҪеңЁе®ҝдё»APPеҶ…иҝҗиЎҢпјҹ

е°ҸзЁӢеәҸзҡ„иҝҗиЎҢе№¶йқһзӣҙжҺҘеөҢе…Ҙе®ҝдё»APPзҡ„дё»иҝӣзЁӢпјҢиҖҢжҳҜйҖҡиҝҮе®ҝдё»APPжһ„е»әзҡ„иҪ»йҮҸзә§е®№еҷЁзҺҜеўғе®һзҺ°пјҢе…¶ж ёеҝғдҫқиө–д»ҘдёӢжҠҖжңҜпјҡ

1.е®№еҷЁеҢ–жһ¶жһ„

  • е®ҝдё»APPдҪңдёәе®№еҷЁпјҡеҫ®дҝЎзӯүе®ҝдё»APPеҶ…зҪ®дәҶдёҖдёӘж··еҗҲжёІжҹ“еј•ж“ҺпјҢйӣҶжҲҗдәҶWebViewпјҲжёІжҹ“и§ҶеӣҫеұӮпјүе’ҢзӢ¬з«Ӣзҡ„JavaScriptеј•ж“ҺпјҲеҰӮJSCoreгҖҒV8пјүиҝҗиЎҢйҖ»иҫ‘еұӮпјҢеҪўжҲҗдёҖдёӘйҡ”зҰ»зҡ„жІҷз®ұзҺҜеўғгҖӮ

  • иө„жәҗйҡ”зҰ»пјҡе°ҸзЁӢеәҸзҡ„д»Јз ҒпјҲWXML/WXSS/JSпјүе’Ңиө„жәҗж–Үд»¶пјҲеӣҫзүҮгҖҒй…ҚзҪ®ж–Үд»¶пјүиў«е°ҒиЈ…дёәзӢ¬з«ӢеҢ…пјҢз”ұе®ҝдё»APPдёӢиҪҪеҗҺеӯҳеӮЁдәҺз§Ғжңүзӣ®еҪ•пјҢдёҺе®ҝдё»APPиҮӘиә«иө„жәҗйҡ”зҰ»гҖӮ

2. еҸҢзәҝзЁӢжЁЎеһӢдёҺйҖҡдҝЎжңәеҲ¶

  • йҖ»иҫ‘еұӮпјҲApp ServiceпјүпјҡиҝҗиЎҢеңЁзӢ¬з«Ӣзҡ„JavaScriptзәҝзЁӢдёӯпјҢиҙҹиҙЈж•°жҚ®еӨ„зҗҶгҖҒAPIи°ғз”Ёе’Ңз”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶпјҢж— жқғзӣҙжҺҘж“ҚдҪңUIгҖӮ

  • и§ҶеӣҫеұӮпјҲView Layerпјүпјҡз”ұWebViewжёІжҹ“йЎөйқўпјҢдҪҶйҖҡиҝҮNativeз»„д»¶жЎҘжҺҘе®һзҺ°й«ҳжҖ§иғҪжёІжҹ“пјҲеҰӮең°еӣҫгҖҒи§Ҷйў‘зӯүеҺҹз”ҹз»„д»¶пјүгҖӮ

  • NativeеұӮдҪңдёәдёӯд»ӢпјҡйҖ»иҫ‘еұӮе’Ңи§ҶеӣҫеұӮзҡ„йҖҡдҝЎйҖҡиҝҮе®ҝдё»APPзҡ„**NativeеұӮпјҲJSBridgeпјү**дёӯиҪ¬пјҢж•°жҚ®з»ҸеәҸеҲ—еҢ–еҗҺдј йҖ’пјҢйҒҝе…ҚзӣҙжҺҘи·ЁзәҝзЁӢи®ҝй—®гҖӮ

йӮЈдёәд»Җд№ҲиҰҒи®ҫи®ЎжҲҗеҸҢзәҝзЁӢжһ¶жһ„е‘ўпјҹйҰ–е…ҲжҲ‘д»¬жқҘеӣһйЎҫдёҖдёӢжөҸи§ҲеҷЁзҡ„зәҝзЁӢжЁЎеһӢпјҢжөҸи§ҲеҷЁжҳҜдёҖдёӘеҚ•зәҝзЁӢжһ¶жһ„пјҢдё»иҰҒеҺҹеӣ жҳҜjsе…Ғи®ёи®ҝй—®ж“ҚдҪңDOMпјҢеӣ жӯӨjsзәҝзЁӢе’ҢжёІжҹ“зәҝзЁӢеҸӘиғҪдә’ж–ҘиҝҗиЎҢгҖӮ

йӮЈе°ҸзЁӢеәҸеҸҲжҳҜеҰӮдҪ•еҒҡеҲ°еҸҢзәҝзЁӢзҡ„е‘ўпјҢж №жң¬еҺҹеӣ е°ұжҳҜеҫ®дҝЎе°ҸзЁӢеәҸзҰҒжӯўjsж“ҚдҪңDOMгҖӮ

дҪҝз”ЁеҸҢзәҝзЁӢжһ¶жһ„зҡ„дјҳеҠҝдёҖзӣ®дәҶз„¶пјҡ

  • жҸҗй«ҳз”ЁжҲ·дҪ“йӘҢпјҲuiе’ҢйҖ»иҫ‘еҲҶзҰ»пјҢйҒҝе…ҚйЎөйқўй•ҝж—¶й—ҙйҳ»еЎһе’ҢеҚЎйЎҝпјү
  • дјҳеҢ–еә”з”ЁжҖ§иғҪпјҲиҝҗиЎҢеңЁдёҚеҗҢзҡ„зәҝзЁӢдёӯпјҢеҸҜд»ҘеҗҢж—¶жёІжҹ“жҲ–иҖ…и®Ўз®—пјү
  • ејҖеҸ‘ж•ҲзҺҮжӣҙй«ҳпјҲи§ЈиҖҰе’Ңжқҫж•ЈиҖҰеҗҲпјү

3. APIд»ЈзҗҶдёҺжқғйҷҗжҺ§еҲ¶

  • NativeиғҪеҠӣд»ЈзҗҶпјҡе°ҸзЁӢеәҸи°ғз”Ёж‘„еғҸеӨҙгҖҒең°зҗҶдҪҚзҪ®зӯүзі»з»ҹеҠҹиғҪж—¶пјҢе®һйҷ…йҖҡиҝҮе®ҝдё»APPзҡ„Nativeд»Јз Ғе®һзҺ°пјҢиҖҢйқһзӣҙжҺҘи®ҝй—®зі»з»ҹAPIгҖӮ

  • жқғйҷҗжІҷз®ұпјҡе®ҝдё»APPеҜ№е°ҸзЁӢеәҸзҡ„жүҖжңүAPIи°ғз”ЁиҝӣиЎҢжқғйҷҗж ЎйӘҢпјҢдҫӢеҰӮжңӘжҺҲжқғзҡ„е°ҸзЁӢеәҸж— жі•и®ҝй—®з”ЁжҲ·йҖҡи®ҜеҪ•гҖӮ

дәҢ. дёәд»Җд№ҲжңүзӢ¬з«ӢиҝӣзЁӢпјҹ

е°ҸзЁӢеәҸеңЁжүӢжңәдёӯжҳҫзӨәдёәвҖңзӢ¬з«ӢиҝӣзЁӢвҖқпјҢжң¬иҙЁжҳҜе®ҝдё»APPйҖҡиҝҮж“ҚдҪңзі»з»ҹжңәеҲ¶пјҲеҰӮAndroidзҡ„WebViewиҝӣзЁӢйҡ”зҰ»гҖҒiOSзҡ„WKWebViewеӨҡиҝӣзЁӢжЁЎеһӢпјүе®һзҺ°зҡ„иҝӣзЁӢзә§йҡ”зҰ»гҖӮе…·дҪ“еҺҹеӣ еҰӮдёӢпјҡ

1. зЁіе®ҡжҖ§дҝқйҡң

  • еҙ©жәғйҡ”зҰ»пјҡиӢҘе°ҸзЁӢеәҸзҡ„WebViewжҲ–йҖ»иҫ‘зәҝзЁӢеҙ©жәғпјҢе®ҝдё»APPзҡ„дё»иҝӣзЁӢдёҚеҸ—еҪұе“ҚпјҲдҫӢеҰӮеҫ®дҝЎиҒҠеӨ©еҠҹиғҪд»ҚеҸҜжӯЈеёёдҪҝз”ЁпјүгҖӮ
  • еҶ…еӯҳйҷҗеҲ¶пјҡзӢ¬з«ӢиҝӣзЁӢзҡ„еҶ…еӯҳз”ұзі»з»ҹеҚ•зӢ¬еҲҶй…ҚпјҢйҒҝе…Қе°ҸзЁӢеәҸеҶ…еӯҳжі„жјҸжӢ–ж…ўе®ҝдё»APPгҖӮ

2. е®үе…Ёеўһејә

  • жІҷз®ұйҡ”зҰ»пјҡе°ҸзЁӢеәҸзҡ„JavaScriptд»Јз ҒиҝҗиЎҢеңЁеҸ—йҷҗзҺҜеўғдёӯпјҢж— жі•и®ҝй—®е®ҝдё»APPзҡ„з§Ғжңүж•°жҚ®пјҲеҰӮеҫ®дҝЎзҡ„иҒҠеӨ©и®°еҪ•пјүгҖӮ
  • зі»з»ҹиө„жәҗз®ЎжҺ§пјҡзӢ¬з«ӢиҝӣзЁӢзҡ„жқғйҷҗз”ұе®ҝдё»APPеҠЁжҖҒжҺҲдәҲпјҢдҫӢеҰӮзҪ‘з»ңиҜ·жұӮйңҖйҖҡиҝҮе®ҝдё»д»ЈзҗҶпјҢйҳІжӯўжҒ¶ж„ҸиЎҢдёәгҖӮ

3. жҖ§иғҪдјҳеҢ–

  • е№¶иЎҢжёІжҹ“пјҡи§ҶеӣҫеұӮпјҲWebViewпјүе’ҢйҖ»иҫ‘еұӮпјҲJSеј•ж“ҺпјүеҸҜиҝҗиЎҢеңЁдёҚеҗҢCPUж ёеҝғпјҢеҮҸе°‘дё»зәҝзЁӢйҳ»еЎһгҖӮ
  • иө„жәҗдјҳе…Ҳзә§пјҡзі»з»ҹеҸҜдёәе°ҸзЁӢеәҸиҝӣзЁӢеҲҶй…ҚзӢ¬з«Ӣдјҳе…Ҳзә§пјҢйҒҝе…Қе®ҝдё»APPзҡ„еҠЁз”»гҖҒдәӨдә’иў«жҠўеҚ иө„жәҗгҖӮ

дёүгҖҒж“ҚдҪңзі»з»ҹзә§е®һзҺ°з»ҶиҠӮ

1. Androidе№іеҸ°

  • еӨҡиҝӣзЁӢWebViewпјҡAndroid 8+ й»ҳи®Өе°ҶWebViewиҝҗиЎҢеңЁзӢ¬з«ӢиҝӣзЁӢпјҲcom.google.android.webviewпјүпјҢе®ҝдё»APPйҖҡиҝҮBinder IPCдёҺе°ҸзЁӢеәҸиҝӣзЁӢйҖҡдҝЎгҖӮ

  • жёІжҹ“иҝӣзЁӢйҡ”зҰ»пјҡжҜҸдёӘе°ҸзЁӢеәҸзҡ„WebViewе®һдҫӢеҸҜиғҪеҲҶй…ҚиҮідёҚеҗҢиҝӣзЁӢпјҢиҝӣдёҖжӯҘйҡ”зҰ»йЈҺйҷ©гҖӮ

2. iOSе№іеҸ°

  • WKWebViewеӨҡиҝӣзЁӢпјҡiOSзҡ„WKWebViewй»ҳи®ӨиҝҗиЎҢеңЁзӢ¬з«Ӣзҡ„com.apple.WebKitиҝӣзЁӢпјҢдёҺе®ҝдё»APPпјҲеҰӮеҫ®дҝЎпјүйҡ”зҰ»гҖӮ

  • JavaScriptCoreеј•ж“ҺпјҡйҖ»иҫ‘еұӮзҡ„JavaScriptд»Јз ҒиҝҗиЎҢеңЁе®ҝдё»APPзҡ„зӢ¬з«ӢзәҝзЁӢдёӯпјҢиҖҢйқһзӢ¬з«ӢиҝӣзЁӢпјҢдҪҶйҖҡиҝҮGCDйҳҹеҲ—е®һзҺ°зәҝзЁӢзә§йҡ”зҰ»гҖӮ

еӣӣгҖҒжҖ»з»“пјҡжһ¶жһ„дёҺзі»з»ҹзҡ„еҚҸеҗҢи®ҫи®Ў

е°ҸзЁӢеәҸзҡ„иҝҗиЎҢдҫқиө–е®ҝдё»APPдёҺж“ҚдҪңзі»з»ҹзҡ„ж·ұеәҰеҚҸдҪңпјҡ

  • 1.е®ҝдё»APPжҸҗдҫӣе®№еҷЁпјҡйӣҶжҲҗеҸҢзәҝзЁӢеј•ж“ҺгҖҒAPIжЎҘжҺҘе’Ңиө„жәҗз®ЎзҗҶгҖӮ

  • 2.ж“ҚдҪңзі»з»ҹжҸҗдҫӣйҡ”зҰ»жңәеҲ¶пјҡйҖҡиҝҮиҝӣзЁӢ/зәҝзЁӢйҡ”зҰ»дҝқйҡңе®үе…ЁжҖ§е’ҢзЁіе®ҡжҖ§гҖӮ

  • 3.жҖ§иғҪдёҺдҪ“йӘҢе№іиЎЎпјҡзүәзүІйғЁеҲҶзҒөжҙ»жҖ§пјҲеҰӮж— DOMж“ҚдҪңпјүпјҢжҚўеҸ–жҺҘиҝ‘еҺҹз”ҹAPPзҡ„жөҒз•…дҪ“йӘҢгҖӮ

3. е°ҸзЁӢеәҸеҲҶеҢ…

дёҖгҖҒеҲҶеҢ…еҹәзЎҖжҰӮеҝө

  • дё»еҢ…вҖӢвҖӢпјҡеҢ…еҗ«еҗҜеҠЁйЎөйқўгҖҒTabBarйЎөйқўе’Ңе…¬е…ұиө„жәҗ
  • еҲҶеҢ…вҖӢвҖӢпјҡд»…еҢ…еҗ«дёҺеҪ“еүҚеҲҶеҢ…зӣёе…ізҡ„йЎөйқўе’Ңз§Ғжңүиө„жәҗ
  • еҲҶеҢ…дјҳеҠҝ
    • вҖӢдјҳеҢ–еҗҜеҠЁйҖҹеәҰвҖӢвҖӢпјҡйҰ–ж¬ЎеҸӘйңҖеҠ иҪҪдё»еҢ…
    • вҖӢвҖӢзӘҒз ҙдҪ“з§ҜйҷҗеҲ¶вҖӢвҖӢпјҡи§ЈеҶі2MBеҚ•еҢ…йҷҗеҲ¶
    • вҖӢжҸҗеҚҮеҚҸдҪңж•ҲзҺҮвҖӢвҖӢпјҡдёҚеҗҢеӣўйҳҹеҸҜиҙҹиҙЈдёҚеҗҢеҲҶеҢ…

дәҢгҖҒеҲҶеҢ…жһ¶жһ„и®ҫи®ЎеҺҹзҗҶ

1. жЁЎеқ—еҢ–йҡ”зҰ»жңәеҲ¶

  • дҪңз”ЁеҹҹжҺ§еҲ¶вҖӢвҖӢпјҡ

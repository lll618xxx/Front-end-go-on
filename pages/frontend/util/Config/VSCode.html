1、设置中文语言

（1）使用快捷键组合【Ctrl+Shift+p】，在搜索框中输入“configure display language”，回车
（2）修改 locale.json 文件下的属性“locale”为“zh-CN”
（3）重启 vscode

​  如果重启后仍然是英文显示，在商店中查看已安装的插件
​  搜索 Chinese（Simplied） Lang 重新安装，然后再重启 vscode

2、单击文件预览取消（原来文件被替换）

利用快捷键 Ctrl+Shift+P， 搜索 settings(json)进入 settings.json 文件

"workbench.editor.enablePreview": false

3、像 sublime 快速将当前行复制到下一行

文件 — 首选项 — 键盘快捷方式
搜索 Shift+Alt+Down 改成 Shift+Ctrl+D

4、快速生成 H5 模板的设置

文件 — 首选项 — 新增用户代码片段 — 输入片段名字 — enter 进入 .code-snippets 文件
在该文件写入如下内容：

{
  "h5 sample": {
    "prefix": "h",
    "body": [
      "<!DOCTYPE html>",
      "<html lang=\"zh-CN\">",
      "<head>",
      "\t<meta charset=\"UTF-8\">",
      "\t<meta name=\"viewport\" content=\"width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=0,minimal-ui:ios\">",
      "\t<title>Document</title>",
      "</head>",
      "<body>\n$3",
      "</body>",
      "</html>"
    ],
    "description": "The full sample code - html5."
  }
}
